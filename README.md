# OpenJDK PlayGround

> **Warning**
> The repo is not maintained anymore. To see the updated playground,
> visit [kotlin-mpp-playground](https://github.com/sureshg/kotlin-mpp-playground)

#### Install OpenJDK

```bash
$ curl -s "https://get.sdkman.io" | bash
$ sdk i java openjdk-ea-24
```

#### Build

```bash
$ ./gradlew build
```
